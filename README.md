# LogManager
Инструмент для анализа, фильтрации и экспорта логов, построенный по MVVM.

# Функционал
1. Поддержка загрузки текстовых файлов с логами;
2. Автоматический парсинг строк с использованием регулярных выражений;
3. Поддержка формата логов [Дата/время] Уровень: Сообщение.
4. Поиск по тексту и фильтрация по уровню (INFO, WARN, ERROR);
5. Группировка записей по уровню и по дате.
6. Возможность экспорта отфильтрованных логов в CSV.
